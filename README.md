# 🤖 Бот для изучения португальских глаголов

## Файлы проекта
- `bot.py` — основной код бота
- `verbs.json` — база глаголов (можно пополнять!)
- `requirements.txt` — список библиотек
- `railway.toml` — настройки для Railway

---

## 🚀 Шаг 1: Загрузка на GitHub

1. Зайди на [github.com](https://github.com) и зарегистрируйся (если нет аккаунта)
2. Нажми **"New repository"** (зелёная кнопка)
3. Название: `verbos-bot`
4. Оставь **Public**, нажми **"Create repository"**
5. На следующей странице нажми **"uploading an existing file"**
6. Перетащи все 4 файла (`bot.py`, `verbs.json`, `requirements.txt`, `railway.toml`)
7. Нажми **"Commit changes"**

---

## 🚀 Шаг 2: Деплой на Railway

1. Зайди на [railway.app](https://railway.app)
2. Нажми **"Start a New Project"**
3. Выбери **"Deploy from GitHub repo"**
4. Авторизуй Railway в GitHub, выбери репозиторий `verbos-bot`
5. Railway начнёт деплой — подожди 1-2 минуты

---

## 🔑 Шаг 3: Добавить переменные окружения

В Railway после деплоя:
1. Нажми на свой проект
2. Перейди в **"Variables"**
3. Добавь три переменные (кнопка **"New Variable"**):

| Имя переменной | Значение |
|---|---|
| `TELEGRAM_TOKEN` | токен от @BotFather |
| `ANTHROPIC_API_KEY` | ключ sk-ant-... |
| `YOUR_TELEGRAM_ID` | твой Telegram ID |

4. После добавления переменных Railway перезапустит бота автоматически

---

## ✅ Шаг 4: Проверка

Открой Telegram, найди своего бота и напиши `/start`

Если всё работает — ты увидишь приветственное сообщение!

---

## 📚 Команды бота

| Команда | Что делает |
|---|---|
| `/start` | Приветствие и список команд |
| `/hoje` | Получить новые глаголы сегодня |
| `/exercicio` | Начать упражнение прямо сейчас |
| `/progresso` | Посмотреть прогресс |

**Расписание (автоматически):**
- 🌅 9:00 — два новых глагола + все формы + примеры
- 🌙 20:00 — упражнение со всеми глаголами

---

## ➕ Как добавить новые глаголы

Открой файл `verbs.json` на GitHub (кнопка карандаша для редактирования).

Скопируй этот шаблон и заполни:

```json
{
  "verb": "ИНФИНИТИВ",
  "translation_ru": "перевод на русский",
  "type": "irregular",
  "theme": "тема",
  "tenses": {
    "presente": ["eu", "tu", "ele", "nós", "vós", "eles"],
    "preterito_perfeito": ["eu", "tu", "ele", "nós", "vós", "eles"],
    "preterito_imperfeito": ["eu", "tu", "ele", "nós", "vós", "eles"]
  },
  "examples": [
    {"pt": "Пример на португальском.", "ru": "Перевод."},
    {"pt": "Ещё пример.", "ru": "Перевод."}
  ]
}
```

После сохранения Railway перезапустит бота автоматически.
